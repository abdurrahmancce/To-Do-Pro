# 📝 To-Do App Pro

A modern, responsive, and feature-rich To-Do application built using **HTML, CSS, and Vanilla JavaScript**.  
This project helps users organize daily tasks with drag & drop functionality, task filtering, localStorage persistence, dark/light mode, priority management, and smooth UI interactions.

---

## 🚀 Live Demo

🔗 [View Live Project](https://abdurrahmancce.github.io/To-Do-Pro/)

---

## ✨ Features

### ✅ Task Management
- Add new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as completed or pending

### 🎯 Productivity Features
- Drag & drop task reordering
- Task filtering:
  - All
  - Pending
  - Completed
- Search tasks instantly
- Task priority levels:
  - Low
  - Medium
  - High
- Due date support
- Progress tracking system

### 💾 Data Persistence
- Saves all tasks using **localStorage**
- Stores:
  - Task list
  - Task order
  - Theme preference

### 🌗 Modern UI/UX
- Glassmorphism design
- Dark & Light mode
- Responsive layout
- Smooth animations
- Toast notifications
- Beautiful empty state
- Keyboard shortcuts support

# ⌨️ Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl + N` | Focus new task input |
| `Ctrl + K` | Focus search input |
| `Delete` | Delete selected task |
| `Enter` | Add or save task |
| `Esc` | Close modal |
